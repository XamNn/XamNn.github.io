Next, let's discuss syntax.

## Numbers
Numbers are all rationals.
```
1.234
```

## Strings
Self explanitory.
```
"my string"
```

### Multiline
**note:** First and last newlines are ignored.
```
`
My multiline string\n
`
```

## Variables
Variables start with lower case. They can be strings, numbers, other variables, function calls, multiline strings, booleans, and more.
```
my_var = "my string"
my_other_var = my_var
```

## Comments
Comments are single line and start with '#'.
```
Print("Hello") # My comment
```

## Math
Mathematics, you know the drill. Add, Sub, Mul, Div, and Mod are supported.
```
x + y
x - y
x * y
x / y
x % y
```

## Dictionaries
Dictionaries act as objects.
```
my_dict               = {}
my_dict.x             = 1 # String key
my_dict["string key"] = 2 # Also string key
my_dict[1]            = 3 # Number key
```

## Lists
Lists are just dictionaries with number keys.
```
my_list = [1, 2, 3]
Print(my_list[2])
```

### Assign
You can assign and perform math on the same line.
```
# These are equal:
x = x + 1
x += 1
```

## Boolenas
Booleans are true or false.
```
my_bool = true
equate        = x == 2
not_equal     = x != 2
less_than     = x < 2
more_than     = x > 2
less_or_equal = x <= 2
more_or_equal = x >= 2
not           = not bool_x
and           = bool_x and bool_y
or            = bool_x or bool_y
```

## If Else
Conditional, the code executes if the boolean is true
```
if x == y {
    Print("x == y")
} else if y == z {
    Print("y == z")
} else {
    print("no match")
}
```

## While
Loop while condition is true.
```
while Condition() {
    Print("I'm inna loop")
}
```

## Functions
Functions first letter is a capital. Programs start in a 'Main' function.
```
function Main() {
    Print(OtherFunction())
}

function OtherFunction() {
    return "Hello!"
}
```

### Functions passed into functions
Functions can be passed into other functions.
```
Function(OtherFunction)
```

### Nested functions
Functions can be nested.
```
function fn1() {
    var = 1
    function fn2() {
        var = 2
    }
    fn2()
    Print(var)
}
