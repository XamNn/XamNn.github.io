Hello there! In this series we are going to develop an interpreted programming language, Interp.

## Goals
The language should be:
1. Simple.
2. Easy to learn.
3. Dynamic. No static typing.
4. Not object-oriented. Structs supported but no classes.
5. All numbers are rational.

## Hello, world
Hello world would look something like this:
```
function Main() {
    Print("Hello, world!")
}
```
